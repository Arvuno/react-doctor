# Contributing to React Doctor

Thanks for your interest in contributing! Please follow these guidelines when submitting changes.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/react-doctor.git`
3. Install dependencies: `pnpm install`
4. Create a feature branch: `git checkout -b pr/mega-campaign-01/add-pkg-metadata`

## Making Changes

- Run tests: `pnpm test`
- Lint: `pnpm lint`
- Type-check: `pnpm typecheck`
- Format: `pnpm format`

## Submitting a Pull Request

1. Ensure all checks pass locally
2. Push your branch to your fork
3. Open a PR against the upstream repository
4. Describe your changes clearly
